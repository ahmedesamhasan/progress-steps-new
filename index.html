What changed

  • No more global selectors tied to one form.
  • Each component instance manages its own state.
  • Auto-init works with data-progress-steps.
  • You can also create instances manually from JavaScript.
  • Custom callbacks are supported for step change, submit, and reset.

Instance API

goToStep(stepNumber)
next()
previous()
reset()
getFormData()

Quick usage

const card = document.querySelector('[data-progress-steps]');
const formFlow = new ProgressStepsForm(card, {
  onSubmit: ({ data }) => console.log(data),
});
